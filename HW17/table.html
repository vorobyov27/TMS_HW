Основные тэги HTML. Парные
Tag Description Info
<html> Основной тэг, в котором
содержится весь HTML
<head> Тэг заголовка, обычно не содержит информации, которая отображается на странице Включает ссылки на css, js файлы, title и тп
<body> Основное видимое тело документа
<style> Содержит CSS стили Используется в <head>
<script> Содержит JS код
  • Используется в <head>.
  • Атрибут scr может использоваться для стороннего JS файла.
<div> Просто отдельный блок Как и <p> располагается на новой строке
<p> Блок абзаца Имеет стандартные отступы, в отличие от <div>
  • <iframe>
  • <frame>
 Фрейм, можно назвать отдельным HTML файлом Код в элементе выполняется независимо
от основного
<table> Таблица
  • Все внутренние теги парные.
  • Заголовки таблицы находятся в тэге <thead>, тело таблицы – в <tbody>.
  • Каждая строка - внутри <tr> (строка заголовков в том числе) – находятся в thead или tbody.
  • В <tr>: ячейка заголовка - <th>. Ячейка таблицы - <td>
<ul><li></li></ul> Список
  • Список может быть внутри списка.
  • ul - список, каждый пункт списка - li
  • <h1><h2><h3>
  • <h4><h5><h6>
Залоловки Заголовок h1 может быть только 1 на странице
<title> Название страницы
  • Используется в <head>.
  • Отображает название страницы на вкладке браузера
<button> Кнопка
  • Альтернатива <input type=“button”>, <input type=“submit”>.
  • Текст между тегами будет названием кнопки.
<select> Комбобокс
  • Каждый элемент комбобокса располагается внутри тэга <select> и имеет тэг <option>Название опции</option>.
  • Атрибут selected - выбранный элемент по-умолчанию.
